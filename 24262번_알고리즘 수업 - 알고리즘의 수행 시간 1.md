### 문제 풀이 핵심 아이디어 :
    ```python
        MenOfPassion(A[], n) {
            i = [n / 2]
            return A[i]; # 코드1
        }
    ```
    입력 크기 n이 주어질 때 알고리즘 수행 횟수와,
    이를 다항식으로 나타냈을 때 최고차항의 차수 출력


### 코드의 시간복잡도와 그 이유 :
    상수 연산을 1번 수행하기 때문에 수행 횟수는 1, 최고차항은 0이 됨.


### 코드 :
```python
print("1")
print("0")
```
