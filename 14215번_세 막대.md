### 문제 풀이 핵심 아이디어 :
        삼각형 성립 조건 : 가장 큰 변이 나머지 두 변의 합보다 작아야 함.
        -> 크다면 두 변의 합에서 1 만큼 빼준 값을 가장 큰 변으로 재정의 후 둘레 측정하기

### 시간복잡도 :
        O(nlogn) : 정렬 라이브러리 씀.
### 코드 :
```python
a = list(map(int, input().split()))
a.sort()

if a[0] + a[1] <= a[2] :
    a[2] = a[0] + a[1] - 1

print(sum(a))
```
