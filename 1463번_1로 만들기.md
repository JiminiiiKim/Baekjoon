### 문제 풀이 핵심 아이디어 :
        동적 프로그래밍으로 모든 숫자의 경우를 구한 다음,
        전에 값을 이용하여 다음 값들을 구함. 
        1) 1 빼기
        2) 3으로 나누기  
        3) 2로 나누기
        모든 경우해서 최솟값 저장하기

### 시간복잡도 :
        O(N) : N의 수만큼 for문을 반복함.

### 코드 :
```python
n = int(input())

dp = [0] * (n+1)

for i in range(2, n+1) :
    dp[i] = dp[i-1] + 1
    if i % 3 == 0 :
        dp[i] = min(dp[i], dp[i // 3] + 1)
